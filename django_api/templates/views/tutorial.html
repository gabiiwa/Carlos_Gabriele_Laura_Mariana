Criar postagem e fazer comentários.

Todas as estudantes podem criar postagens contendo curiosidades, divulgação de eventos, oportunidades de estágio/programas, entre outras coisas. Inclusive, é possibilitado comentários em cada postagem, permitindo uma maior interação. Para a criação de postagem, basta acessar a página "Criar postagem" e preencher todos os campos. Para fazer algum comentário: acesse a postagem, disponível na "Home"; clica no botão "Fazer comentário"; preenche o campo do texto do comentário; e, por fim, clique em "Comentar".

Títulos e Ranking

Para incentivar a interação das participantes, este sistema é gamificado devido às faixas de pontuação por meio de títulos, nomeados com grandes mulheres da área de computação. A medida que as estudantes se pontuam (criar postagem, fazer comentários, ler postagem), avançam nos títulos para que possam ser vistas como referências em outras estudantes, principalmente que estão començando a vida universitária. Para visualizar seu título atual, basta acessar a página "Título Atual". É possibilitado a visuallização de todos os usuários, na página "Usuários" e todos os usuários que possui o mesmo título com o seu, na página "Usuários com o mesmo título".
